1. Назначение ПО (что может приложение)
Программа hardload.exe предназначена для копирования некоторого количества файлов 
разного формата и размера из одной папки в другую

2. Условия использования ПО
Программа предназначиена для работы на платформе Windows всех версий
3. Подготовка к работе
Программа не требует какой либо предварительной подготовки. Так как файл hardload.exe является единственным.
4. Работа с системой
При запуске программы Hardload.exe нас встречает главный экран приложения
На нем расположены интуитивно понятные кнопки управления
 -Кнопка с тремя точками в верхнем левом углу открывает проводник, для выбора нужного файла для копирования
выбор производится двойным нажатием на нужный файл или нажатием кноппки "открыть"
 -Кнопка с тремя точками в нижнем правом углу открывает окно выбора директории, куда нужно скопировать файл
выбор производится нажатием на нужную папку и далее кнопки "ОК"
 -При выборе нужных файлов, они отображаются в списке на главном экране приложения
 -Для копирования их нужно выделить галочкой (слева)
 -Кнопка начала копирвоания, зеленый треугольник в верхней строке. Соответственно начинает копирование
 -Кнопка паузы, справа от кнопки начала копирования, останавливвает процесс копирования 
 -Кнопка удаления, красный крестик. Удаляет добавленные файлы из списка. Для удаления нужно не ставить галочку,
а нажать на путь к этому файлу, выделить его в списке синей полосой. И нажать на кнопку удаления

6. Ограничения
Нельзя скопировать файл, не выделив его галочкой в списке
Нельзя удалить файл из списка, не выделив его в списке синей полосой 
Так же нельзя удалить файл выделив его галочкой или нажав del на клавиатуре 
Нельзя скопировать папку целиком
Нельзя добавить в список сразу несколько файлов одновременно, только по одному
Отсутствуют уведомления об ошибках и завершении копирвоания
При копирвоании файлов боьшого объема наблюдается зависание программы, но файл копируется. Нужно подождать
На Windows 11, для корректного отображения текста, нужно использовать английский язык
